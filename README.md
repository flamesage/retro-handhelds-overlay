# Retro Handhelds Overlay

An overlay for Retro Handhelds Discord server.

![Application Screenshot](./Screenshot.png)

## How can I run this?

### 1. Download the latest release

Choose the ZIP file for your platform:
- **Windows** → [Download ZIP](https://github.com/flamesage/retro-handhelds-overlay/releases/latest/download/retro-handhelds-overlay-windows.zip)
- **Linux** → [Download ZIP](https://github.com/flamesage/retro-handhelds-overlay/releases/latest/download/retro-handhelds-overlay-linux.zip)
- **macOS** → [Download ZIP](https://github.com/flamesage/retro-handhelds-overlay/releases/latest/download/retro-handhelds-overlay-macos.zip)

### 2. Extract the ZIP

- **Windows**
  - Right-click the ZIP → **Extract All…**
- **macOS**
  - Double-click the ZIP
- **Linux**
  - Extract the ZIP file

### 3. Run the application

#### Windows
Double-click: `retro-handhelds-overlay.exe`

#### Linux
```sh
chmod +x retro-handhelds-overlay
./retro-handhelds-overlay
```

#### macOS
```sh
chmod +x retro-handhelds-overlay
./retro-handhelds-overlay
```

If macOS blocks the app:
1. Open **System Settings → Privacy & Security**
2. Scroll down and click **Allow Anyway**
3. Run the app again

### 4. Configure

Edit `settings.toml` in the same folder to customize the overlay.

Changes take effect the next time you launch the application.
